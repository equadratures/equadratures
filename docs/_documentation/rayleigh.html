Rayleigh¶

The Rayleigh distribution.

class equadratures.distributions.rayleigh.Rayleigh(scale)[source]¶

The class defines a Rayleigh object. It is the child of Distribution.

Parameters

scale (double) – Scale parameter of the Rayleigh distribution.

getCDF(points=None)[source]¶

A Rayleigh cumulative density function.

Parameters

  • self (Rayleigh) – An instance of the Rayleigh class.

  • points (array) – Points at which the CDF needs to be evaluated.

Returns

Cumulative density values along the support of the Rayleigh distribution.
