Matrix factorization utilities¶

Utilities with QR factorization.

equadratures.qr.bidiag(A)[source]¶

Computes the bidiagonalization of the m-by-n matrix A := U B V’, where m >=n. Here U is an m-by-n orthogonal matrix, B is a n-by-n bi-diagonal matrix and V is a n-by-n orthogonal matrix.

Parameters:matrix A (numpy) – an m-by-n matrix
Returns:U, the left orthogonal matrix
Return type:numpy matrix
Returns:B, the bidiagonal matrix
Return type:numpy matrix
Returns:V, the right orthogonal matrix
Return type:numpy matrix

** Notes ** Uses the algorithm of Golub and Kahan (1965) and requires 4mn^2 - 4n^3/3 flops.

equadratures.qr.directElimination(A, b, C, d)[source]¶

Solves the constrained least squares problem min ||Ax-b||_2 subject to Cx=d via the direct elimination method. :param numpy ndarray A: an m-by-n A matrix :param numpy ndarray b: an m-by-1 b matrix

Returns:x, the coefficients of the least squares problem.
Return type:ndarray
Returns:cond, the condition number of the final matrix on which least squares is performed
Return type:float
equadratures.qr.house(vec)[source]¶

Returns a scalar and a vector that may be used to form a Householder matrix

Parameters:ndarray vec (numpy) – The input vector that needs to be reflected in the hyperplane spanned by the Householder vector, v
Returns:v, the Householder vector
Return type:ndarray
Returns:beta, the Householder scalar
Return type:double
Notes
The Householder vector is given by P = I - beta * v * v’
