Gamma¶

The Gamma distribution.

class equadratures.distributions.gamma.Gamma(shape=None, scale=None)[source]¶

The class defines a Gamma object. It is the child of Distribution.

Parameters

  • shape (double) – Shape parameter of the gamma distribution.

  • scale (double) – Scale parameter of the gamma distribution.
