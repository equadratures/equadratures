Beta¶

The Beta distribution.

class equadratures.distributions.beta.Beta(lower=None, upper=None, shape_A=None, shape_B=None)[source]¶

The class defines a Beta object. It is the child of Distribution.

Parameters:
  • a (double) – First shape parameter of the beta distribution. This value has to be greater than 0.
  • b (double) – Second shape parameter of the beta distribution. This value has to be greater than 0.
  • lower (double) – Lower bound of the support of the beta distribution.
  • upper (double) – Upper bound of the support of the beta distribution.
