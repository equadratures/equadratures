Convex optimization utilities¶

A library of convex optimizers.

References:

  • Joshi, S., Boyd, S., (2009) Sensor selection via convex optimization, IEEE Transactions on Signal Processing, 57(2). Paper.

  • Candes, E., Romberg, J., (2005) L1-Magic: Recovery of sparse signals via convex programming. Paper.

equadratures.convex.CG_solve(A, b, max_iters, tol)[source]¶

Solves Ax = b iteratively using conjugate gradient, assuming A is a symmetric positive definite matrix.

Parameters

  • A (numpy-matrix) – The matrix.

  • b (numpy-array) – The right hand side column vector.

  • max_iters (int) – Maximum number of iterations for the conjugate gradient algorithm.

  • tol (double) – Tolerance for cut-off.

equadratures.convex.bp(A, b, x0=None, cgtol=None, cgmaxiter=None, pdtol=None, pdmaxiter=None, verbose=False, use_CG=False)[source]¶

Solving the noiseless basis pursuit problem.

Parameters

  • A (numpy-matrix) – The matrix.

  • b (numpy-array) – The right hand side column vector.

  • x0 (numpy-array) – Initial solution; if not provided the least norm solution is used.

  • cgtol (double) – Tolerance for conjugate gradients.

  • cgmaxiter (int) – Maximum number of iterations for the conjugate gradient method.

equadratures.convex.maxdet(Amatrix, k)[source]¶

Determinant maximization routine.

Parameters

  • Amatrix (numpy-matrix) – A matrix with more columns than rows.

  • k (int) – The number of column subsamples required such that the determinant of the resulting matrix is maximized.

Notes

This code is based on Newton’s method as detailed in Joshi and Boyd (2009).
