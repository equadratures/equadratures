Download Links¶

To download the code, please visit our GitHub page. Click either on the Fork Code button or Clone or download.

Installation: For installation on Mac and Linux systems, simply type the following into the terminal.

sudo python setup.py install

For installation on Windows, you will need Anaconda; select the Python 2.7 option. Upon successful installation, open the Sypder IDE and go into the Effective-Quadratures-master directory and type the following in the command window

python setup.py install

This should build the code. Just make sure you include the location of effective_quadratures folder to your python file and you should be good to go. To run this code you will require python 2.7, numpy, scipy and matplotlib.
